# svelte-loading-spinners

> collection of loading spinners with Svelte.js

## Installation

```bash
npm i svelte-loading-spinners
```

or

## Usage

```svelte
  <div id="app">
    <Jumper></Jumper>
  </div>

<script>
  import {Jumper} from 'svelte-loading-spinners'
</script>
```
